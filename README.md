## Description

Template Frontend / Backend (Vuejs / Nesjs)
Api data nosql

## Installation

installation des dépendances globales
```bash
$npm install
```
ensuite installation de l'ensemble des dépendances Nestjs / Vuejs

```bash
$ npm install install:api-vue
```

## Running the app

```bash
# development
$ npm run serve:dev


# production mode
$ npm run start:prod 
```
